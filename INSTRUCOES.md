# Instruções para instalar os pets

Escolha um dos métodos:

- [Instalação manual](INSTRUCOES-INSTALACAO-MANUAL.md)
- [Instalação com o instalador portátil para Windows](INSTRUCOES-INSTALADOR.md)

O método manual funciona no Windows, macOS e Linux. O instalador portátil
funciona somente no Windows 10 e Windows 11.
